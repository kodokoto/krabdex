# krabdex workspace

This workspace hosts two crates:
- [**krabdex**](https://crates.io/crates/krabdex): type-safe, async Rust SDK for the PokeAPI.
- [**krabdex-mcp**](https://crates.io/crates/krabdex-mcp): MCP server exposing krabdex tools for Codex-compatible agents.

See per-crate READMEs for details:
- SDK: [crates/krabdex/README.md](crates/krabdex/README.md)
- MCP server: [crates/krabdex-mcp/README.md](crates/krabdex-mcp/README.md)
