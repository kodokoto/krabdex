# krabdex workspace

This workspace hosts two crates:
- **krabdex**: type-safe, async Rust SDK for the PokeAPI.
- **krabdex-mcp**: MCP server exposing krabdex tools for Codex-compatible agents.

See per-crate READMEs for details:
- `crates/krabdex/README.md` — SDK usage, examples, errors, development.
- `crates/krabdex-mcp/README.md` — MCP server setup, Codex config, tools list.
